1.1 使用 Goland

Goland 是 JetBrains 公司推出的 Go 语言 IDE，是一款功能强大，使用便捷的产品。

下载地址：http://www.jetbrains.com/go

对于 Mac 和 Windows，都有直接的安装文件，直接双击后，傻瓜式安装，一路 next，直到完成。

1.2 使用 atom

安装好 atom 工具，然后安装 go-plus 插件和 atom-terminal-panel 等插件。

1.安装 go-plus 插件，这个插件提供了 Atom 中几乎所有 go 语言开发的支持，包括 tools, build flows, linters, vet 和 coverage tools。它还包含很多代码片段和一些其它特性。
